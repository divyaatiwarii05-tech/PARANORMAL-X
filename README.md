# PARANORMAL-X

## AI-Powered Paranormal Investigation Simulator

### Project Overview

PARANORMAL-X is a final-year BCA software engineering project that simulates professional paranormal investigations using Artificial Intelligence, Internet of Things (IoT), and a Java-Python architecture.

The system allows investigators to:

- Investigate haunted locations
- Collect paranormal evidence
- Use scientific investigation tools
- Generate AI-based ghost personalities
- Produce automatic investigation reports
- Connect real IoT sensors (ESP32)
- Analyze paranormal cases
